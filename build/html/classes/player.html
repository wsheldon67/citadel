Player¶

Represents a player in the game. This class has a lot of useful methods that are often the best ones to use to accomplish something. I would not recommend making changes to this class unless you need to, see if what you want can be accomplished by changing will_citadel.Game or one of the will_citadel.Entity subclasses.
